These Java packages are used in jOAI, a OAI-PMH data provider and harvester Java web application. Note that a number of unused Java packages remain here that were part of the Digital Discovery System (DDS) and the Digital Catalog System (DCS) webapps, which have since been moved to a separate code repository.

This software is developed by Digital Learning Sciences (DLS) at the University Corporation for Atmospheric Research (http://www.ucar.edu/).

See the Expression Language Function Library descriptions. These functions are for use in JSP pages and JSTL tags.
